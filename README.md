# Medium Crawler

## Installation

Install requirements

```
$ pipenv install
```

Install requirements for developing packages

```
$ pipenv install --dev
```

## Configurable variables

|  variable name                |   default value       | overridable |
| ----------------------------- | --------------------- | ----------- |
| PROXY                         | http://127.0.0.1:8787 | YES         |
| PROXY_ENABLED                 | FALSE                 | YES         |
| PROXY_DEPTH                   | 0                     | YES         |
